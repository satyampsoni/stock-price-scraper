# Stock Price Scraper

A Go-based stock price scraper that fetches and stores real-time stock prices from various financial data sources for analysis and record-keeping.


## Features

- Scrapes real-time stock prices.
- Saves data in CSV format for easy analysis.
- Lightweight and built with Go for performance and efficiency.

## Requirements

- Go 1.16 or higher
- Active internet connection to access live stock data.
